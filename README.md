# ZPlatform

## SetUp Project

- Clone project

```sh
$ git clone https://github.com/nshaphan/zplatform-ui.git
$ cd zplatform-ui
```

- Install Dependencies

```sh
$ yarn

```

- Create a .env file in root folder and add the following content

```env
REACT_APP_API_URL=http://<host>:<port>
```

- Run development Server

```sh
yarn start

```
